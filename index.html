Código do Google Apps Script

📋 Instruções de Configuração:

  1. Crie DUAS planilhas no Google Sheets (uma para Análises Simples, outra para Comparativos)
  2. Em cada planilha, vá em Extensões → Apps Script
  3. Apague todo o código existente e cole o código abaixo
  4. Salve (Ctrl+S) e clique em Implantar → Nova implantação
  5. Selecione App da Web
  6. Configure: Executar como "Eu" e Acesso "Qualquer pessoa"
  7. Clique em Implantar e copie a URL gerada
  8. Cole a URL no arquivo index.html na seção CLOUD_CONFIG
